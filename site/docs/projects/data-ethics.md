# Data Ethics Paper

Source folder: `Data-Ethics/`

## Summary

Written project reviewing three readings on data ethics, public trust, and responsible big-data practice.

## Analysis Used

- Comparative review of international data-governance approaches.
- Discussion of the illusory truth effect and how repeated framing affects trust.
- Synthesis of responsible-research guidelines from "Ten Simple Rules for Responsible Big Data Research".
- Practical reflection on transparency, privacy, accountability, and communication.

## Technologies and Methods

- Literature review and written argumentation
- Comparative policy/ethics analysis
- LaTeX document preparation (`main.tex`, `coverPage.tex`)
