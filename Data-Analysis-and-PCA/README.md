# Data Analysis and PCA

Homework 2 used the UCI Wisconsin Breast Cancer dataset to practice preprocessing, dimensionality reduction, and simple modeling.

- Replaced placeholder "?" values, dropped incomplete and duplicate records, and normalized the target label into a binary class.
- Standardized feature columns and explored both manual target normalization and pandas factorize for label encoding.
- Ran PCA to study variance explained and to visualize class separation after scaling.
- Trained a k-NN classifier on cleaned data and evaluated with accuracy, confusion matrices, and classification reports.
