# Gradient Descent

Homework 3 explored gradient descent behavior on custom cost functions with interactive visualizations.

- Defined scalar objective functions and analytical derivatives, then validated outputs.
- Visualized cost curves and highlighted starting points using Plotly for intuition about curvature and minima.
- Experimented with step sizes and iterations to illustrate convergence and oscillation patterns.
- Included reusable plotting helpers to inspect trajectories and compare parameter settings.
