# Data Science Fundamentals Portfolio

Coursework and projects from my Data Science Fundamentals class. The capstone piece is **Profiling Academic Trajectories**, a full ML exploration of student performance data with clustering, classification, and interpretability.

## Projects

- [Profiling-Academic-Trajectories](Profiling-Academic-Trajectories/README.md) — Largest project; clustered and classified higher-ed student performance using UCI’s Higher Education dataset, plus SHAP-driven feature insights and an internal codebook of categorical encodings.
- [Life-Expendancy-Analysis](Life-Expendancy-Analysis/README.md) — WHO life expectancy deep dive with cleaning, country standardization, exploratory visuals, clustering, and PCA for dimensionality reduction.
- [Data-Analysis-and-KNN](Data-Analysis-and-KNN/README.md) — k-NN classification on the Palmer penguins dataset with data cleaning, feature engineering, and evaluation metrics.
- [Data-Analysis-and-PCA](Data-Analysis-and-PCA/README.md) — PCA-driven exploration of the UCI Wisconsin Breast Cancer dataset, including preprocessing, scaling, and model performance checks.
- [Gradient-Descent](Gradient-Descent/README.md) — Gradient descent exercises with custom visualizations of cost functions and step-by-step optimization behavior.
- [Data-Ethics](Data-Ethics/README.md) — Written analysis of three data ethics articles covering governance, illusory truth effects, and responsible big-data research.
- [Lecture-Notes](Lecture-Notes/README.md) — Lecture notebooks and sample datasets used throughout the course.
