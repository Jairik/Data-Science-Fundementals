# Profiling Academic Trajectories

Capstone project analyzing the UCI Higher Education Students Performance dataset.

- Fetched the dataset programmatically, paired it with an internal codebook (see dataset-docs.md) to decode categorical encodings, and set an ethics plan for handling socioeconomic attributes.
- Cleaned and profiled 32 demographic, socioeconomic, and academic features; built train/test splits for predictive tasks.
- Clustered students to explore behavioral patterns, then trained classification models to predict GPA/grade categories.
- Used SHAP-based interpretability and feature importance plots to explain model decisions and highlight high-impact study habits.
