# Data Ethics Paper

Essay reviewing three readings on data ethics and integrity.

- Compared governance approaches for data ethics across countries and reflected on gaps between policy and practice.
- Discussed the illusory truth effect and how repetition and presentation can distort public trust in data stories.
- Summarized "Ten Simple Rules for Responsible Big Data Research" and weighed transparency against privacy concerns.
- Argued for practitioner accountability, clear context, and honest communication as core responsibilities for data scientists.
